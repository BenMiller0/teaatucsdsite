Interactive Audio Animatronic Encounter

An engaging project using audio-driven robotics to create a dynamic themed entertainment experience.

Project Overview

This project blends real-time audio processing and mechanical engineering to bring characters to life. Through voice-controlled servo motors and custom-designed robotics, students simulated mouth movements that match speech, enhancing the realism of a themed encounter experience.

Key Features

  • Real-time voice recognition and amplitude analysis
  • Precise servo motor control for animatronic mouth movement
  • Custom-built character "Taro" for public demonstrations
  • Collaborative design across engineering and storytelling disciplines
