The Force in Your Hands

A Star Wars Inspired Experience

Coming soon.

Wield the Force without touching a thing. Use intuitive, real-time gesture controls powered by machine learning and immersive technology to move objects and save the galaxy.

Project Overview

Using Raspberry Pi technology and lightweight machine learning models, this project introduces intuitive gesture-based interactions for attractions. Guests control elements of the experience with simple hand motions, enhancing immersion and creating an unforgettable Star Wars-inspired journey.

Key Features

  • Real-time Force gesture detection
  • Highly optimized machine learning model
  • Control lighting, sound, and animatronic features
  • Seamless themed integration for maximum immersion
